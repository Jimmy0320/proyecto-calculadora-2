Funciones Básicas de la Calculadora

  • Suma: Permite sumar dos o más números introducidos.
  • Resta: Permite restar un número de otro.
  • Multiplicación: Permite multiplicar dos o más números.
  • División: Permite dividir un número entre otro.
  • Borrar (C): Permite borrar la entrada actual de los números y operaciones.
  • Resultado (=): Permite obtener el resultado de la operación matemática.

¿Para qué fue creada esta calculadora?

La Calculadora Básica fue desarrollada como parte de un proyecto educativo dentro del curso de Diseño Web, con el objetivo de proporcionar una herramienta sencilla y eficiente para realizar operaciones matemáticas.

Este proyecto tiene como fin ayudar a los estudiantes del Técnico Laboral en Análisis y Desarrollo de Sistemas a aplicar y reforzar sus conocimientos sobre el desarrollo de aplicaciones web, proporcionando una experiencia práctica en el uso de HTML, CSS y JavaScript.

Es una excelente herramienta para aquellos que desean practicar habilidades matemáticas básicas o necesitan realizar cálculos rápidos en su día a día.
